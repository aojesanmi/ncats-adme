Background

ADME@NCATS is a resource developed for the scientific community by NCATS to host in silico prediction models for different ADME endpoints. The resource allows prediction of different pharmacokinetic properties of small molecules which is potentially useful for structure optimization in early drug discovery. Users can provide input as a CSV/Text file containing SMILES. Alternatively, the users can sketch a molecule using the editor provided. For each compound, the predictions from the models are provided as output along with the confidence scores. Those users who cannot upload their data are encouraged to download the self-contained instance of ADME@NCATS that can be run locally.

A subset of the data that were used for developing the models are provided for download on the individual model pages. The users are encouraged to use these data for development and validation of future QSAR models.
